一. if-else

格式

if(布尔值表达式) //如果为true则执行
    程序主干
else if(布尔值表达式) //如果为true则执行
    程序主干
else //上述全为false时执行

例程

public class IfElse {
    static int result = 0;
    static void test(int a,int b) {
        if (a > b) {
            result = +1;
        }
        else if (a < b) {
            result = -1;
        }
        else {
            result = 0;
        }
    }
    public  static void main(String[] args){
        test(10,5);
        System.out.println(result); //+1
        test(5,10);
        System.out.println(result); //-1
        test(2,2);
        System.out.println(result);  //0
    }
}

二.While

迭代语句

此语句会重复执行，知道其控制作用的布尔值为false时方会停止执行

格式

while(boolean值){
    程序主干
}

例程

    int i = 1; //循环变量初始化
    while( i <= 10 ) {//循环条件,当i大于10时，此时布尔值转为false,循环停止。
        System.out.println(i);//执行语句
        i++;
    }
/*
因为布尔值固定为true,故此时为死循环。
    while(true) {
        
    }
 */

三.do-while

do-while与while的区别就是do-while中的语句至少执行一次，即便起控制作用的布尔值第一次就是flase.

格式

do 
    程序主体 
while(布尔值表达式)
    程序主干

四.for

这种循环第一次会进行初始化，然后进行条件测试，每次迭代结束后，还会有某种形式的“步进”

初始化和步进部分都是按顺序执行的，初始化会最先执行

格式

for(初始化表达式; 布尔值表达式; 步进(可空))
    程序主体

例程

public class ForTest {
  public static void main(String args[]) {
    int i;

    for (i = 0; i < 3; i = i + 1)
      System.out.println(i);
  }
}
/*输出
0
1
2
*/

六.return

用途：

1.指定一个方法的返回值（不存在就返回void）

2.使当前方法退出，并返回这个值

注意：

1.如果一个返回void值的方法后面没有return,则它后面会有一个隐含的return

2.返回非void值的方法，则必须确定每一条代码路径都会返回一个值

例程

public class ReturnTest {
    static int test(int a, int b) {
        if (a > b) {
            return +1; //满足条件，返回 +1
        }
        else if (a < b) {
            return -1; //满足条件，返回 -1
        }
        else {
            return 0; //满足条件，返回 0
        }

    }
    public  static void main(String[] args) {

        System.out.println(test(10,5));//+1
        System.out.println(test(5,10)); //-1
        System.out.println(test(2,2));  //0
    }
}

七.break和continue

1.break用于跳出一个循环体或者完全结束一个循环 (结束了其所在的整个循环)

2.continue语句的作用是跳过本次循环体中剩下尚未执行的语句，立即进行下一次的循环条件判定 ** (只是跳过了符合其条件的一次**循环)

Break例程(达成条件结束循环时用)

条件达成时，该循环彻底停止，不会继续执行

public class Test3 {
    public static void main(String[] args) {
        for(int i=0; i<12; i++){
            if(i==5){
                break;//到 i==5 ,这一个for循环直接全部结束
            }
            System.out.print(i+" ");
        }
    }
}
/*输出
0 1 2 3 4
*/

Continue例程(跳一次的时用)

条件达成时，停止该循环中符合条件的那一次循环，后继循环将继续运行

public class Test4 {
    public static void main(String[] args) {
        for(int i=0; i<12; i++){
            if(i==5){
                continue; //有且仅有跳过了5这一次的循环，整体循环还将继续
            }
            System.out.print(i+" ");
        }
    }
}
/*输出
0 1 2 3 4 6 7 8 9 10 11
*/

五.Switch循环(分支结构)

Switch在后继JAVA版本中提供了更加简化的语法，这个笔记先不展开，黑马的笔记里有提及。

句式

格式说明：

1.整数值选择器：(将生成用于匹配的值，与下面case的值进行匹配)取值为byte，short，int，char等整数值，JAVA7后对于非整数值加入了字符串->String.

2.**case **:后面跟的是要和表达式进行比较的值（被匹配值）。

3.break :表示中断，结束的意思，用来结束switch语句,没有的话它会继续执行知道遇到下一个break,俗称case穿透

4.default :表示所有情况都不匹配的时候，就执行此处的内容，和if语句的else相似。(其执行完毕的地方是所有break跳转的目的地)

5.case后面的值只能是字面量，不能是变量

6.case给出的值不允许重复

switch(表达式){
    case 值1:
        语句体1;
        break;
    case 值2:
        语句体2;
        break;
    ....
     default:
        语句体n+1;
        break;
}

例程：

    需求：键盘录入星期数，显示今天的减肥活动。
    周一：跑步
    周二：游泳
    .....

整数例程(注:字符串例程详见基础卷P96-5.9例程)

import java.util.Scanner;

public class Test9 {
    public static void main(String[] args) {
      

        //分析：
        //1.键盘录入星期数
        Scanner sc = new Scanner(System.in);
        System.out.println("请录入星期");
        int week = sc.nextInt();
        //2.利用switch对星期进行匹配
        switch (week){
            case 1:
                System.out.println("跑步");
                break;
            case 2:
                System.out.println("游泳");
                break;
         /*
                .........(省略)
                                         */
            default:
                System.out.println("没有这个星期");
                break;
        }
    }
}
